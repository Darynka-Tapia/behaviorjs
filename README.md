# 🛡️ Behavior.js (Core Tracker)

Librería de **UX Analytics** para medir la intención y fricción del usuario mediante el análisis de interacciones en tiempo real.

### 🎯 Objetivos del Proyecto
* **Métricas de Tiempo:** Desglose de `Visual Discovery` (búsqueda) y `Cognitive Hesitation` (duda).
* **Auditoría de Affordance:** Análisis automático de estilos CSS para evaluar la "clicabilidad" de los elementos.
* **Filtro de Inactividad:** Medición de tiempo activo real excluyendo periodos de *Idle*.

### 📊 Captura de Datos
El sistema monitorea automáticamente los elementos marcados con `data-behavior` y registra:
- **TTA Detallado:** Tiempos de reacción y DNI del elemento.
- **Miss-clicks:** Clics fallidos fuera de zonas de interacción.
- **Scroll Tracking:** Hitos de lectura (25%, 50%, 75%, 100%).
